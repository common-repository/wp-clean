== Description ==

Плагин WP-Clean предоставляет большое количество функций для оптимизации, безопасности и удаления лишних файлов (кода) в WordPress.

На странице настроек «Виджеты», можно:
* отключить редактор блоков Gutenberg на странице виджетов;
* отключить виджет «Архивы»;
* отключить виджет «Календарь»;
* отключить виджет «Рубрики»;
* отключить виджет «Мета»;
* отключить виджет «Страницы»;
* отключить виджет «Свежие комментарии»;
* отключить виджет «Свежие записи»;
* отключить виджет «RSS»;
* отключить виджет «Поиск»;
* отключить виджет «Облако меток»;
* отключить виджет «Текст»;
* отключить виджет «Произвольное меню»;
* отключить виджет «HTML-код»;
* отключить виджет «Аудио»;
* отключить виджет «Видео»;
* отключить виджет «Галерея»;
* отключить виджет «Изображение»;
* отключить виджет «Блок».

На странице настроек «Безопасность», можно:
* включить перенаправление с http на https;
* отключить переход по ссылке /?author=;
* скрыть версию WordPress;
* удалить ?ver у файлов стилей;
* удалить ?ver у файлов скриптов;
* отключить Trackback на собственный сайт;
* удалить rsd ссылку;
* удалить wlwmanifest ссылку;
* отключить XML-RPC и удалить ссылку X-Pingback;
* удалить файл license.txt;
* удалить файл readme.html;
* удалить файл install.php;
* удалить файл wp-config-sample.php;
* изменить текст ошибки авторизации;
* отключить возможность сброса пароля для учётных записей;
* запретить вход на страницу «Забыли пароль?».

На странице настроек «Header», можно:
* отключить «JSON REST API»;
* отключить «Emoji»;
* удалить «dns-prefetch»;
* удалить «jquery-migrate.min.js».

На странице настроек «Профиль пользователя», можно:
* запретить изменение цветовой гаммы;
* не показывать «admin bar» на лицевой части сайта.

На странице настроек «Комментарии», можно:
* удалить поле «Сайт» в форме комментариев;
* удалить имя пользователя администратора из CSS-классов в комментариях;
* удалить ссылки ?replytocom в комментариях;
* запретить HTML-код в комментариях;
* удалить ссылку на автора комментария.

На странице настроек «Интерфейс», можно:
* удалить блок «Помощь»;
* удалить блок «Настройки экрана»;
* удалить ссылки на сайт WordPress из «admin bar»;
* удалить текст «Спасибо вам за творчество с WordPress»;
* удалить номер версии WordPress;
* скрыть иконки в боковой колонке;
* удалить уведомление «Ваш браузер устарел»;
* удалить уведомление «Требуется обновление PHP».

На странице настроек «Записи», можно:
* отключить редактор блоков Gutenberg для всех типов записей;
* отключить автоматическое сохранение записи;
* удалить canonical ссылку записи;
* удалить ссылку shortlink у записи;
* удалить ссылки adjacent_posts_rel_link у записи.

На странице настроек «Поиск», можно:
* отключить поиск на сайте;
* исключить страницы из результатов поиска.

На странице настроек «Вложения», можно:
* при удалении записи также удалять все вложения;
* удалить страницы вложений.

На странице настроек «Изображения», можно:
* отключить функцию SRCSET;
* разрешить WordPress загружать SVG файлы.

На странице настроек «Архивные страницы», можно:
* отключить «Архив автора»;
* отключить «Архив по тегам»;
* отключить «Архив по дню»;
* отключить «Архив по месяцу»;
* отключить «Архив по году»;
* отключить «Архив по категориям».

На странице настроек «Обновления», можно:
* отключить авто-обновление «ядра»;
* отключить авто-обновление «тем оформления»;
* отключить авто-обновление «плагинов»;
* отключить авто-обновление «переводов»;
* запретить обновление «ядра»;
* запретить обновление «тем оформления»;
* запретить обновление «плагинов».

На странице настроек «RSS ленты», можно:
* отключить формат ленты RSS2;
* отключить формат ленты Atom;
* отключить формат ленты RDF;
* удалить ссылки на дополнительные RSS-ленты сайта;
* удалить ссылки на основные RSS-ленты сайта.

На странице настроек «Постраничная навигация», можно:
* закрыть страницы пагинации от индексации;
* удалить заголовок блока постраничной навигации.

P.S: Если у вас остались какие-либо вопросы пишите на почту <cite>info[@]artemsannikov.ru</cite>.

== Installation ==
	
1. Загрузите плагин "WP Clean" в директорию `/wp-content/plugins/`;
2. Активируйте "WP Clean" на странице `Плагины` в панели управления.
3. Перейдите в настройки плагина и установите галочку напротив функций, которые необходимо активировать.
